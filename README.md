Ladder
======

An ELO ladder implementation built with Grails.

Plugins used:
- spring-security-core:1.2.7.3
- twitter-bootstrap:2.0.2.25
- fields:1.1
- remote-pagination:0.3
- resources:1.2-RC1
- cached-resources:1.0
- zipped-resources:1.0
- yui-minify-resources:0.1.5
- cache-headers:1.1.5
